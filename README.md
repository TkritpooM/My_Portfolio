# 🌐 Personal Portfolio – React + Vite

A clean, modern, and responsive portfolio website built with **React**, **Vite**, and **Tailwind CSS**.  
Showcasing my **projects**, **skills**, and **contact information** with animated UI and a sleek dark theme.

---

## 🚀 Tech Stack

- ⚛️ **React + Vite** – Fast and modern frontend setup  
- 🎨 **Tailwind CSS** – Utility-first CSS framework  
- 🎞️ **Framer Motion** – Smooth animations and transitions  
- ☁️ **Deployed on Vercel**
